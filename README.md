<h3>Description</h3>

This project is a web application that visualizes COVID-19 data in different regions of Brazil.

The data was obtained from https://infoms.saude.gov.br/extensions/covid-19_html/covid-19_html.html (official Brazilian government website).

The dashboard features data reading and provides insights into COVID-19 in Brazil within a Flask development environment.
